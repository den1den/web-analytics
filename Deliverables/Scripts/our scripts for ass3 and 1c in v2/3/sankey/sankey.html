Below are the interactive versions of our sankey diagrams. Beware, to make it interactive it uses a Google Chart libary that takes some time to load (approx 10 seconds). We found out that the bigger versions are on the limit of what the Chart library can process. So when there are problems you should try to use another browser, like Firefox (v44.0) on Linux. After the loading is done the graph will be displayed.
